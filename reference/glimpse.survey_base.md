# Get a glimpse of a survey design object

Print a transposed summary of the survey object's columns — column names
run down the left, data types and values run across. Respects
[`select()`](https://dplyr.tidyverse.org/reference/select.html): if
columns have been selected, only those columns are shown; design
variables are hidden from the display.

## Usage

``` r
glimpse.survey_base(x, width = NULL, ...)
```

## Arguments

- x:

  A
  [`survey_base`](https://jdenn0514.github.io/surveycore/reference/survey_base.html)
  object.

- width:

  Width of the output. Defaults to the console width.

- ...:

  Passed to
  [`dplyr::glimpse()`](https://pillar.r-lib.org/reference/glimpse.html).

## Value

`x` invisibly.

## See also

[`select()`](https://dplyr.tidyverse.org/reference/select.html) to
control which columns are visible

Other selecting:
[`pull.survey_base()`](https://jdenn0514.github.io/surveytidy/reference/pull.survey_base.md),
[`relocate.survey_base()`](https://jdenn0514.github.io/surveytidy/reference/relocate.survey_base.md),
[`select.survey_base()`](https://jdenn0514.github.io/surveytidy/reference/select.survey_base.md)
